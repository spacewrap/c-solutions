### Exercise 6.05
Which one of the following statements is not equivalent to the other two
(assuming that the loop bodies are the same)?

(a) `while (i < 10) {`...`}`  
(b) `for (; i < 10;) {`...`}`  
(c) `do {`...`} while (i < 10);`

### Answers
(c). If `i` starts at > 10, the do loop is executed once. In (a) and (b), the loops are skipped entirely.
