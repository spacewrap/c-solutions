Latin-1 characters (You'll need to look up the Latin-1 code points for these
characters). For example, the phrase *déjà vu* could be represented by the
string `"d\xe9j\xe0 vu"`.

(b) *crème brûlée*  
(c) *crème fraîche*  
(d) *Fahrvergnügen*
